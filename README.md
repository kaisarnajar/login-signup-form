# Login & Signup Form

A simple and responsive Login & Signup form built using HTML and CSS. This project focuses on creating a clean and modern user interface for authentication pages.

## Features

* Responsive design
* Login form
* Signup form
* Modern and clean UI
* Beginner-friendly project structure

## Project Structure

```text
login-signup-form/
│
├── index.html
├── style.css
└── README.md
```

## Getting Started

### Clone the repository

```bash
git clone <repository-url>
cd login-signup-form
```

### Run the project

Simply open `index.html` in your preferred web browser.

No additional setup or dependencies are required.

## Technologies Used

* HTML5
* CSS3

This project was created to practice frontend development concepts, including form design, layout creation, and responsive styling using HTML and CSS.
